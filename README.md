# adflowgram
인스타그램 클론 (with Express + Prisma + React and React Native)








# setting

## graphql-yoga 설치
```
> yarn add graphql-yoga
```

## nodemon 설치
- 파일을 저장할때 마다 서버를 재실행 해주는 도구
```
> yarn add nodemon -D
```

## babel-node 설치
```
> yarn add babel-node
> yarn add babel-cli -D
> yarn add @babel/preset-env @babel/node @babel/core
```

## dotenv 설치
- 서버를 세우는 툴(?)
```
> yarn add dotenv
```
